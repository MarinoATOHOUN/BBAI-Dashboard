# **App Name**: BlackBenAI Admin Dashboard

## Core Features:

- Employee Management: CRUD operations for employees with roles, salaries, and status management.
- Document Management: Secure document upload, encryption, and access control based on user roles using Fernet or AES encryption.
- Project Tracking: Management of internal projects with assigned members, descriptions, and status tracking.
- Internal Announcements: Publish and manage internal announcements accessible to all employees.
- Financial Management: Track incoming and outgoing finances with graphical representation for analysis.
- User Authentication and Authorization: Secure login/logout/register functionality via JWT with role-based permissions (Admin, RH, Employee, Manager).
- Dashboard Analytics: AI-powered tool to extract, summarize and highlight key trends and insights from the admin panel data. Trends include employee performance, project success rates and budget adherence.

## Style Guidelines:

- Primary color: Emerald green (#22c55e), inspired by growth and sophistication, complements a dark theme.
- Background color: Dark charcoal (#0b0b0b) to reflect a sleek, futuristic tech environment.
- Accent color: Electric blue (#00b3ff) for interactive elements and highlights, analogous to the primary for visual interest.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Use a consistent set of minimalist icons from 'lucide-react' for navigation and actions.
- Fixed side navbar and minimal header layout for consistent navigation and efficient use of screen space.
- Subtle transitions using 'framer-motion' for a fluid and engaging user experience.
